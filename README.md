## Installation

```bash
git clone https://github.com/TTomilin/DIC.git
cd DIC
```

### Conda (recommended)
```bash
conda create -n demo python=3.11
conda activate demo
```

### Run env
```bash
pip install .
```

## Run

```
python scripts/1_run_gym_cartpole.py
```
